Widget Test

Simple test to verify widget functionality

Test Instructions

1. Click "Open Widget" - Widget should appear in bottom-right corner

2. Try typing in the input field and pressing Enter

3. Try dragging the widget around by clicking the header

4. Try resizing by dragging the bottom-right corner

5. Click "Close Widget" to hide it

Debug Info

Check browser console for debug messages

Widget should be draggable and resizable

Input field should accept text and respond to Enter key
